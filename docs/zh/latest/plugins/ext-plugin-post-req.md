## 描述

`ext-plugin-post-req` 插件的功能与 `ext-plugin-pre-req` 插件类似。

唯一不同的是：它在内置 Lua 插件执行之后且在请求到达上游之前工作。

参考文档 [ext-plugin-pre-req](./ext-plugin-pre-req.md) 去学习如何配置并使用它。
